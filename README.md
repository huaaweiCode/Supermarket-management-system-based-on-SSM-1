# 基于SSM的超市管理系统

#### 介绍
基于SSM的超市管理系统拥有两种角色

管理员：用户管理、商品管理、分类管理、咨询列表管理、权限管理、支付管理、留言管理、角色管理、管理员管理、订单管理、采购管理等

用户：商品查看、购买、购物车、商品下单、设置收货地址、登录注册等
